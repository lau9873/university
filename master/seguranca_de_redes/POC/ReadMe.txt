De modo a poder correr esta simulação são necessários os seguintes pré requisitos:

- OMNET++ instalado.
- Framework INET (Mais recente).
- Framework Veins.
- SUMO.

Antes fazer build ao projeto é necessário importar VEINS e INET para o OMNET++ e nas opções de build fazer uma
referência a ambos.

- Caso esteja a utilizar Linux é necessário fazer um set das variáveis de ambiente com o seguinte comando:

$source setenv

- No caso de ser em ambiente Windows apenas basta fazer build do projecto e correr a simulação com o nome
veins_inet.
